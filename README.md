# Vero

Vero is a blockchain-based digital student identity and payment prototype for UCT.

## Workspace Packages

- `packages/issuer-agent`
- `packages/verifier-agent`
- `packages/admin-portal`
- `packages/wallet`
- `packages/shared`
- `tests/e2e`

## Phase 0 Note

This phase sets up the monorepo structure only. Issuer logic, verifier logic, wallet native setup, admin portal implementation, and end-to-end flows are deferred to their respective phases.
